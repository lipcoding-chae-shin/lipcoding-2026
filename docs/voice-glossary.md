# 음성 코딩 용어집 (Voice Glossary) — 입코딩 2026

> **에이전트에게**: 받아쓰기(STT)는 한국어 발음을 영어 기술 용어로 자주 **오인식**한다.
> 사용자가 말한 단어가 아래 "들리는 말" 과 비슷하면 **"표준 용어/명령"** 으로 해석하라.
> 확신이 80%면 표준 명령을 **복창**하고 진행, 그 미만이면 **한 줄로 되묻는다**.
> 절대 들린 철자 그대로 존재하지 않는 명령을 만들지 마라.

---

## 0. 빠른 규칙

- **동음이의·오인식이 의심되면 표준 명령을 복창**: 예) "디플로이 해줘" → *"Azure Developer CLI 로 `azd up` 배포할게요."*
- 사용자가 **알파벳을 또박또박**(에이-제트-디) 부르면 그게 명령 철자다 → `azd`.
- 한 글자 명령은 보통 **CLI 도구**다: `az`(Azure), `gh`(GitHub), `azd`(Azure Developer).
- 문맥상 이 대회의 기본 축은 **Copilot SDK · Azure OpenAI · azd 배포** 셋이다. 애매하면 이 셋 쪽으로 해석.

## 1. Azure 계열 (가장 자주 틀림 ⚠️)

| 들리는 말(STT 오인식 예) | 표준 용어 | 실제 명령/식별자 |
|---|---|---|
| 애저 / 아주어 / 아저 / 에이저 / 아주 | **Azure** | (클라우드 플랫폼) |
| 애저 씨엘아이 / 애저 시엘아이 / 아주 씨엘아이 / 에이지 / 에이즈 | **Azure CLI** | `az` ("에이-제트") |
| 애저디 / 에이지디 / 에이제트디 / azd / 아즈드 | **Azure Developer CLI** | `azd` (배포는 `azd up`) |
| 애저 오픈에이아이 / 애저 오픈 AI / 오픈에이아이 | **Azure OpenAI** | `AZURE_OPENAI_ENDPOINT` 경유 |
| 파운드리 / 파운더리 / 마소 파운드리 / 에이아이 파운드리 | **Microsoft Foundry (Azure AI Foundry)** | 모델 카탈로그/배포 |
| 앱 서비스 / 애프 서비스 | **Azure App Service** | 웹 호스팅 |
| 컨테이너 앱스 / 컨테이너 앱 | **Azure Container Apps** | 컨테이너 호스팅 |
| 스태틱 웹 앱 / 정적 웹앱 | **Azure Static Web Apps** | 정적+Functions |
| 키 볼트 / 키볼트 / 키 발트 | **Azure Key Vault** | 시크릿 저장 |
| 리소스 그룹 / 리소스그룹 | **Resource Group** | `az group ...` |
| 매니지드 아이덴티티 | **Managed Identity** | 키 없는 인증 |
| 아주 업 / 애저 업 / 에이지디 업 | **`azd up`** | 프로비전+배포 |
| 아주 다운 ⚠️ | **`azd down`** | 리소스 삭제 → **반드시 확인** |

> "애저"가 안 잡히면 사용자가 **"Microsoft 클라우드"** 또는 **"에이-제트"** 라고 바꿔 말하게 안내해도 된다.

## 6. 자주 쓰는 "의도 → 표준 동작" 매핑 (대회 특화)

받아쓰기로 들어온 자연어 지시를 이 표준 동작으로 정규화한다.

| 사용자가 말하면 | 표준 해석 / 실행 |
|---|---|
| "배포해줘" / "디플로이" / "올려줘" | **`azd up`** (1순위). 첫 배포면 `azd init` → `azd up`. |
| "내려줘" / "지워줘"(리소스) | **`azd down`** → ⚠️ **비가역, 음성 확인 후 실행.** |
| "모델 불러줘" / "AI 붙여줘" | **Azure OpenAI deployment** 호출(외부 모델 금지). |
| "코파일럿으로 ~하게" | **Copilot SDK** 통합 코드 작성(얕은 단발 호출 금지, 흐름에 녹임). |
| "커밋하고 푸시" | `git add -A && git commit && git push` (메시지 복창). |
| "로컬에서 띄워줘" / "돌려봐" | `npm run dev` 후 **브라우저로 실제 동작 확인**. |
| "환경변수 넣어줘" | `.env`(로컬) + Azure App Settings/Key Vault(배포), 시크릿 코드 박지 않기. |
| "문서 확인해줘" | **Microsoft Learn MCP / 공식 문서**로 API 검증(환각 금지). |

## 7. 발음이 안 잡힐 때, 사용자에게 권하는 대체 발화

- **Azure** → "에이-제트" 또는 "마이크로소프트 클라우드"
- **`az`** → "에이-제트 명령"
- **`azd`** → "에이-제트-디" (또는 "Azure Developer CLI")
- **`gh`** → "지-에이치" 또는 "GitHub 명령줄"
- **OAuth vs auth** → 위임이면 "오-auth", 그냥 인증이면 "인증"
- **route vs root** → "url 경로"면 route, "최상위"면 "루트 디렉토리"
- 안 되면 **알파벳을 한 글자씩**: "에이, 제트, 디" → `azd`
